# 858D-Hot-Air-Station-Flat-Wall-Mounted-Case

This project takes any generic 858D hot air station and makes it flat and wall mounted for space effeciency.

You can watch [this](https://youtu.be/YhG3GC7g7KI) YouTube video to learn more!

All the models are uploaded here and at [MakerWorld](https://makerworld.com/en/models/3081193-flat-858d-hot-air-station-mod). If you would like to make this thing your self please download the files from MakerWorld not from here because MakerWorld will mostl likely be the most upto date and they support me with points, github doesnt.
